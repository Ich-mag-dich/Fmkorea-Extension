## `how to use`

- 크롬 확장 프로그램에 추가.

- 게시글 제목 우클릭으로 미리보기.

- Esc키 입력 또는 미리보기 창 밖 클릭으로 닫기.

- 팝업에서 읽은 게시글 쿠키 삭제 가능.

- 미리보기 중 추천 가능 (<https://www.fmkorea.com/{here}>
  here에 해당하는 부분의 경로가 존재하는 페이지에서 미리보기한 게시글만 가능)
